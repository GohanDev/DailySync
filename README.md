# DailySync

Aplicação Android desenvolvida no âmbito da disciplina de Desenvolvimento de Aplicações Móveis.

## Funcionalidades
- Login simples
- Listagem de compromissos
- Adicionar compromisso
- Editar compromisso
- Remover compromisso
- Localização opcional
- Secção "Sobre"

## Tecnologias
- Kotlin
- Retrofit
- REST API própria (Node.js)
- Render (deploy da API)

## API
https://dailysync-api.onrender.com/
